An interactive dashboard that provides analytics on:
- Broader ETFs  
- Sectoral ETFs  
- Thematic ETFs
- Glpbal ETFs

Features:
- ETF explorer  
- Detailed analytics  
- Benchmark insights  
- Category-level charts  
- Filters and search  
- Attractive UI
